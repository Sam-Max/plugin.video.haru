# plugin.video.haru

a basic kodi anime addon

[https://pikdum.github.io/repository.pikdum/](https://pikdum.github.io/repository.pikdum/)

## design goals

- correct
- fast
- simple

## requirements

- kodi 19+
- debrid account

## features

- anime list
- airing anime list
- watched indicators
- batch support
- watch history
- nyaa search
